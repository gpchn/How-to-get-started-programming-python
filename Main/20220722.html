使用技巧

具体的使用方法我也就不多说了，Tkinter简明教程里讲的非常明白，所以这里我就讲一些文中没有提到的小技巧吧。

新式组件外观

其实通过教程里的配图就能看出来，Tkinter的组件外观太老了，不是很好看，作者也是曾经被这个问题困扰了很久。后来找到一种方法，其实Tkinter库里面是自带一套新的组件的，储存在“tkinter.ttk”这个库里，所有组件的名称都是一样的，所以我们只要把tkinter.xxx换成tkinter.ttk.xxx就行了。（这里还有很多种不同的方法，效果都是一样的，作者就不多解释了，毕竟这不是一本入门教程书[doge]）

三种布局方法

在教程中，所有的布局方法都是用的grid()，作者平时用到的布局方法也只有三种：grid()，place()，pack()。

  • grid的使用方法
from tkinter.ttk import Somefunc
xxx = Somefunc(someargs)
xxx.grid(column=0, row=0)

这里可以看到，grid的常用参数只有两个：column和row。

如果学过这两个英语单词，应该可以猜出来，column是竖列，row是横排。

grid会自动根据组件的横竖排列，规划合适的位置。

  • place的使用方法
from tkinter.ttk import Somefunc
xxx = Somefunc(someargs)
xxx.place(x=5, y=5)

不难看出，place的两个常用参数就是x轴和y轴的关系。

place不像grid，它不会自动规划位置，只会根据x轴和y轴参数放置组件，所以你能完全控制组件的位置，这也是作者最爱用的一种方法。

  • pack的使用方法
from tkinter.ttk import Somefunc
xxx = Somefunc(someargs)
xxx.pack()

说真的，pack的参数作者不常用啊，因为作者用它主要是临时排列。

众所周知，tkinter的组件不定义位置是不会显示的，但是每次定义位置（特别是有很多组件的情况下）实在麻烦，所以作者有时候会使用pack进行临时排列，因为它字最少，也不用参数[doge]

至于具体参数的使用方法，请读者们自行上网搜索吧。（作者贴心地给你们一个必应搜索地址：https://cn.bing.com/?q=tkinter.pack）
